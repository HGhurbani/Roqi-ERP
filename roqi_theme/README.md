# ROQI Theme

Theme app for ERPNext (Frappe) that applies ROQI branding (yellow/black), favicon and logo.

- Injects CSS bundles to override primary colors
- Sets website favicon and splash image

## Usage (local bench)

```
bench get-app /path/to/roqi_theme
bench --site yoursite install-app roqi_theme
bench build --app roqi_theme
```
